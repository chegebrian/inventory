# 🖥️ LocalShop Inventory Frontend

A modern **React-based frontend application** for the LocalShop Inventory System. This client-side app provides a clean user interface for managing inventory, interacting with the backend API, and visualizing data.


## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone <your-repo-url>

```

### 2. Navigate to client folder

```bash
cd client
```

### 2. Install Dependencies

```bash
npm install
```

---

## ▶️ Running the Application

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```
